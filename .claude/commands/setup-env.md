Set up the development environment for working on this firmware:

1. Check if Python 3 is installed
2. Check if required build tools are available (gcc-arm-none-eabi, make)
3. Install Python dependencies from requirements.txt and requirements-dev.txt
4. Initialize git submodules if not already done
5. Configure QMK to use this repository: qmk config user.qmk_home=$PWD
6. Verify the setup by running a test build
7. Provide troubleshooting tips if anything fails

This ensures the development environment is properly configured.
